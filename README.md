# yummycli

An AI-friendly CLI for multimodal model providers.

## Status

`yummycli` is being rebuilt from scratch.

## Phase 1 Scope

Phase 1 focuses on:

- Gemini provider support
- API key initialization and status checks
- Image generation
- Image editing
- Provider-first shortcuts for Gemini image flows
- A capability-first internal command structure

## Current Direction

The CLI will expose a provider-first command surface for end users while keeping a capability-first internal architecture for long-term stability.
